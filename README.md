# OrderManagement

Baseline estrutural de uma API REST de pedidos em .NET.

Nesta fase a solution mantém apenas a base técnica para evolução:
- OpenAPI com Scalar
- tratamento de erros com Problem Details
- health check
- EF Core com SQL Server
- projeto de testes preparado, ainda sem regras de domínio

Ainda não existem entidades de pedido, casos de uso ou endpoints reais de negócio.
